
# Roadmap

Status: public beta.

## Current beta goals

- Stabilize the Windows portable bundle.
- Keep the GUI as the primary product surface.
- Keep SQLite as the runtime source of truth.
- Improve first-run setup and diagnostics.
- Improve issue reports through About dialog diagnostics and build metadata.
- Harden extraction/review workflows for Forza Motorsport, 2023 release, post-race Results screen screenshots.

## Next beta candidates

- Better onboarding for LM Studio configuration.
- More visible first-run database setup checks.
- More precise guidance for supported and unsupported screenshot layouts.
- Synthetic or redacted sample material for documentation.
- More public-facing troubleshooting documentation.

## Out of scope for beta

- Bundling LM Studio.
- Bundling model weights.
- Cloud OCR as a default workflow.
- Telemetry collection.
- Forza Horizon support.
- Older Forza Motorsport UI layouts as a product target.
- Mobile or non-Windows packaging.
